## 📋 Deskripsi

**Afrillia Pulsa** adalah aplikasi web pembayaran tagihan multi-layanan dan pengisian pulsa yang dirancang dengan tampilan modern dan estetik. Aplikasi ini mensimulasikan proses pembayaran berbagai tagihan rutin masyarakat Indonesia.

### ✨ Fitur Unggulan

| Fitur | Deskripsi |
|-------|-----------|
| 🔐 **Sistem Login** | Login dengan username `afril` dan password `afril123` |
| 📊 **Dashboard** | Ringkasan saldo, transaksi hari ini, total pengeluaran |
| 💡 **Bayar Tagihan** | PLN, PDAM, Internet, Seminar dengan 3 metode pembayaran |
| 🎓 **SPP** | Cek NIM, pilih cicilan semester, bayar multiple |
| 📱 **Isi Pulsa** | 6 provider (Telkomsel, XL, Indosat, Tri, Smartfren, Axis) |
| 📜 **Riwayat** | Filter, hapus, dan lihat detail transaksi |
| 🌙 **Dark Mode** | Mode gelap/terang dengan toggle |
| 💳 **3 Metode Bayar** | Virtual Account, QRIS (dengan countdown), Teller/Kasir |
| 📄 **Bukti Bayar** | Cetak struk pembayaran |
