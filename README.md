## 📝 Deskripsi Singkat

**Afrillia Pulsa** adalah aplikasi web pembayaran tagihan multi-layanan dan pengisian pulsa yang dirancang dengan tampilan modern dan estetik. Aplikasi ini mensimulasikan proses pembayaran berbagai tagihan rutin masyarakat Indonesia seperti:

- 💡 Listrik (PLN)
- 🚿 PDAM
- 🌐 Internet
- 🎓 Biaya Kuliah (SPP)
- 📱 Isi Pulsa & Paket Data

Dibangun dengan HTML5, CSS3, dan JavaScript murni (Vanilla JS) tanpa backend server. Semua data disimpan di localStorage browser.

## 🚀 Cara Menjalankan

### Cara 1: Langsung Buka File
1. **Download** atau clone repository ini
2. **Ekstrak** file ZIP (jika download ZIP)
3. **Buka** file `index.html` dengan double-click
4. Aplikasi akan terbuka di browser

### Cara 2: Menggunakan Live Server (VS Code)
1. Buka folder project di VS Code
2. Install ekstensi "Live Server"
3. Klik kanan `index.html` → "Open with Live Server"

### Cara 3: Akses Online
Kunjungi link demo: [https://afrillia-pulsa.vercel.app](https://afrillia-pulsa.vercel.app)
